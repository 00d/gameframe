# Alchemist

The alchemist brews potions, bombs, and mutagens using alchemy.

## Key Ability
Intelligence

## Hit Points
8

## Initial Proficiencies
- Perception: Trained
- Saving Throws: Fortitude Trained, Reflex Trained, Will Expert
- Skills: Trained in Crafting, plus additional skills equal to 2 + Int modifier
- Attacks: Trained in simple weapons, unarmed attacks
- Defenses: Trained in light armor, unarmored defense
- Class DC: Trained

## Research Field
Choose a field (e.g., Bomber, Chirurgeon).

## Alchemist Feats
- Alchemical Crafting (1st).
- Quick Alchemy (1st).

## Class Features
- **Alchemy** (1st): Create alchemical items.
- **Research Field** (1st): Specialized abilities.
