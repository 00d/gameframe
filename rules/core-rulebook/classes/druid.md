# Druid

The druid communes with nature, wielding primal magic and shapeshifting.

## Key Ability
Wisdom

## Hit Points
8

## Initial Proficiencies
- Perception: Trained
- Saving Throws: Fortitude Trained, Reflex Trained, Will Expert
- Skills: Trained in Nature, plus additional skills equal to 2 + Int modifier
- Attacks: Trained in simple weapons, unarmed attacks
- Defenses: Trained in light and medium armor, unarmored defense
- Class DC: Trained

## Druidic Order
Choose an order (e.g., Animal, Leaf, Storm).

## Druid Feats
- Wild Shape (1st): Polymorph into animals.
- Primal Spellcasting (1st).

## Class Features
- **Primal Spellcasting** (1st): Spontaneous primal spells.
- **Wild Shape** (1st): Transform into beasts.
